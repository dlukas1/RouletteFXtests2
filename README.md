Simple roulet game made for learning programming on Java.

Project have 3 classes: 
Main - for graphics + counting user money
Roulette - returns random number from 0 to 36 and assign color to number;
Teller - is checking is user win or not;


How to play:
- choose where you like to bet - is it number, color, odd or even?
- choose how much you like to bet.
- press "SPIN" button (ENTER also works)
